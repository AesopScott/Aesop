✗ Current homepage problems → ✓ What the redesign fixes

Current — Problems

15 compliance buttons above the fold
No clear "start here" path for new users
V1 and V2 courses split into separate 185–692KB files
Pedagogy, advisory boards orphaned with no nav home
institutional-procurement.html unreachable from nav
Nav has no dropdown — can't surface deep pages
Hero has 3 competing CTAs (Assessment, V2, V1)
Standards alignment uses hover-cards — mobile-unfriendly

Redesign — Fixes

Compliance moves to /standards — homepage stays clean
Hero has exactly 2 CTAs: Browse Courses + Take Assessment
Courses Hub consolidates V1+V2 under one URL
About page gives pedagogy + advisory boards a home
For Schools page with procurement vehicles prominently linked
Dropdown nav surfaces 20+ pages without cluttering the bar
Content directory on homepage links every page — nothing hidden
Standards as text links — scannable on any device
My Recommendation
Concept 3 (Clean Directory) + this site structure

Concept 3's minimal hero + organized content grid handles both new users (courses dominant, clear CTA) and returning users (everything visible at a glance). Pair it with the structural changes above — move compliance to /standards, consolidate courses, add the dropdown nav — and the homepage goes from a governance document to an actual welcome mat.
